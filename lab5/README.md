# Обучение языку программирования C#

## Лабораторная работа №5

Реализовать обобщенный двунаправленный список и тесты, проверяющие его работоспособность. У списка должна быть следующая функциональность:

1. Поддержка ***интерфейса IEnumerable<>***

2. Поддержка ***интерфейса ICollection<>***

3. ***Функция Insert*** – вставляет элемент перед определенным элементом.

4. ***Свойства First*** и ***Last*** – возвращают первый и последний элементы списка.

5. Список должен поддерживать ***события Added, Removed, Cleared***.
